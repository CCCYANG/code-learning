这个仓库用于记录日常算法练习、LeetCode 刷题，以及一些编程题、笔试题和业务规则题的实现过程。

目前主要以日期目录的方式归档，每个文件对应一道题，代码里会尽量保留题目描述、解题思路、实现代码和必要注释，方便后续复盘。

## 仓库内容

- LeetCode 与算法题练习
- 编程题、笔试题记录
- 字符串、数组、排序、Map、DFS、BFS、区间计算等题型
- 项目级 ESLint 规则与保存自动修复配置

## 代码风格

项目已经配置 ESLint，主要规则包括：

- 使用 2 个空格缩进
- JavaScript / TypeScript 行尾不加分号
- 函数名与括号之间保留空格，例如 `function demo () {}`
- Trae / VS Code 保存时自动执行 ESLint 修复

可以运行：

```bash
npm run lint
```
